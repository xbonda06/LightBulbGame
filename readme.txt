Description:
Light Bulb is a puzzle game where the goal is to connect circuits and light up all the bulbs.
The game supports different difficulty levels (easy, medium, hard).
It also includes an archive where players can view all previously played games and replay them.
Additionally, there is a multiplayer mode that allows playing with 1 to 3 opponents.
